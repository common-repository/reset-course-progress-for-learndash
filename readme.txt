== Description ==
LearnDash’s Reset Course Progress streamlines the process of resetting course progress from your WordPress dashboard. With a single click, you may reset the progress for all or selected courses by users, user roles, and groups. In the progress result tab, it also shows the reset status/result.

[youtube https://www.youtube.com/watch?v=DvffxFfKNLo]

## Features:
**Reset by user(s)**
  You can reset the progress by selecting specific users and their specific courses Or for all courses.
  
**Reset by user role(s):**
  You can reset the progress by selecting specific user roles and specific courses Or for all courses at once.
  
**Reset course progress for specific group(s):**
  You can also reset the progress by selecting specific groups and respective specific courses Or for all group courses.

== Installation ==
= Automatic installation =

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of Reset Course Progress For LearnDash, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type “Reset Course Progress For LearnDash,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.
	
= Manual installation =
	
Manual installation method requires downloading the Reset Course Progress For LearnDash plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Updating =

Automatic updates should work smoothly, but we still recommend you back up your site.
